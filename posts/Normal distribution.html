PMF V/S PDF 1. PMF is for Discrete Data The PMF (Probability Mass Function) applies to discrete random variables, meaning variables that take on countable values (e.g., number of students, number of cars). It gives the probability of exact values occurring.

  1. PDF is for Continuous Data Our dataset consists of continuous stock returns (e.g., real numbers like 0.52, -1.23, 1.89). For continuous distributions, the probability of a single exact value occurring is zero because there are infinitely many possible values. Instead of a PMF, we use a Probability Density Function (PDF) to describe the relative likelihood of values falling within an interval.

Since stock returns are continuous, we cannot use a PMF. Instead, we must use a PDF (such as a normal distribution or kernel density estimation) to describe their distribution.

The Cumulative Distribution Function (CDF) gives the probability that a random variable 𝑋 is less than or equal to a specific value 𝑥, i.e., F(x)=P(X≤x) It helps in understanding the cumulative probability of the data.

cdf=stats.norm.cdf(x_vals, loc=loc, scale=scale)
plt.plot(x_vals, cdf, color="black")
plt.show()

PPF The Percent-Point Function (PPF) is the inverse of the Cumulative Distribution Function (CDF). It is also called the quantile function because it helps find the value of a random variable at a given probability.

In simple terms, if:

𝐹(𝑥)=𝑃(𝑋≤𝑥)

then PPF finds 𝑥 for a given probability 𝑝, i.e., 𝑥=𝐹−1(𝑝)

This helps in determining critical values for a normal distribution.

# Plot the CDF
plt.figure(figsize=(8, 5))
plt.plot(x_vals, cdf_vals, color="black", label="Normal CDF")

# Mark the chosen height on the CDF
plt.vlines(x_mark, 0, cdf_mark, linestyle=":", color="red", label=f"CDF at {x_mark}")
